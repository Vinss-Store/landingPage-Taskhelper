## ✨ Tentang Proyek

**TaskHelper** adalah landing page bisnis jasa dokumen akademik yang dibangun menggunakan **React + Vite**. Halaman ini menampilkan profil bisnis secara lengkap mulai dari Business Model Canvas hingga layanan dan kontak.

Dibuat sebagai bagian dari tugas mata kuliah **Rancangan Bisnis Digital (RBD) — Semester 6**.

## 🚀 Menjalankan Proyek

### Prerequisites
- Node.js versi 18+
- npm atau yarn

### Instalasi

```bash
# Clone repository
git clone https://github.com/Vinss-Store/landingPage-Taskhelper.git

# Masuk ke direktori
cd landingPage-Taskhelper

# Install dependencies
npm install

# Jalankan dev server
npm run dev
```

Buka [http://localhost:5173](http://localhost:5173) di browser.

### Build untuk Production

```bash
npm run build
```

Output akan tersimpan di folder `dist/`.

## 📦 Deploy ke Vercel

### Via Website (Direkomendasikan)
1. Login ke [vercel.com](https://vercel.com) dengan akun GitHub
2. Klik **"Add New Project"** → Import repo ini
3. Vercel otomatis deteksi Vite — klik **Deploy**

### Via CLI
```bash
# Install Vercel CLI (jika belum)
npm i -g vercel

# Deploy
vercel --prod
```
